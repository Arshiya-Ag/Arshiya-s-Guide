What Is Css ?

  • Css (Cascading Style Sheets) Code E Ast Ke Zamani Be Vojod Omad ke Programmer Ha Be In Natije Residan Ke Faghat Ba Html Nmitonan Ye Site Ro Be Zibayi Style Konan. Be Hamin Dalil Css Makhsos inkar Be Vojod Amad.

How We Can Use Css In Our Code ?

  • Baraye Inke Az Code Haye Css Estefade Konim 3 Rah Vojod Darad:

    1. inline

    2. internal

    3. external

1.Inline

  • Dar Ravesh InLine Ma Baraye Har Attribue Ke Mikhahim Taghiri Dar Stylesh Ijad Konim Be Sorati Ke Dar Zir Mibinid Ye Elemend Style Ijad Mikonim Va Taghir Hayi Ke Mikhahim Emal Mikonim

2.Internal

  • Dar Ravesh Internal Ma Miayim Baraye File Html e Ke Mikhahim Style ro Taghir Bedim Dar Ghesmat Head Ye tag style Baz Mikonim Va Taghir Hayi Ke Mikhahim Ro Emal Mikonim.

3.External

  • Dar Ravesh External Ke Maghol Tarin Va Por Estefade Tarin Raveshe Ma Miayim yek folder css Sepas Yek file styles.css Dorost Mikonim Va Code Haro Dar Onja Minevisim Va Hameye File Haye Html Ro Be In File Ba Tag link Be Sorat Zir Vasl Mikonim.

The Box Model Of Website Styling

  • Har Elementi Ke Ma Dar Yek Site Darim Yek Box Ast Ke Az Ghesmat Haye Mokhtalefi Shekl Gerefte Ke Shamel:

    1. Content: Mohtavaye Element Ro Dar Bar Dare.

    2. Padding: Fasele Beyn Mohtavaye Text Va Border Ro Neshan Midahad.

    3. Border: Be Marz Element Gofte Mishavad Mohtavaye Element Daron In Marz Gharar Darad.

    4. Margin: Ba Tanzim In Style Mitoni Fasele Beyn Element Ba Element Haye dige Ya Body ro Tanzim Konim.

CSS Display Property

  • Display Yek Property Ast Ke Hadafash Taghir Halat Neshan Dadan Yek Element Ast. In Property Value Haye Mokhtalefi Darad Ke Alan Mikhahim Ba Mohem Tarin Anha Ashana Shavim.

    1. block: In Value Display Ke Dar Tag Hayi Mesl h1 Vojod Darad Betori Ast Ke Height ro Ta Jayi Ke Niaz Darad Eshghal Mikonad Ama Width ro Be Sorat Kamel Eshghal Mikonad Va Nmizarad Element Dige E Dar Kenarash Beshinad Va Hamintor Mitavan Be In Display Yek Width Moshkhas Ham Dad.

    2. inline: In Value Display Ke Dar Tag Hayi Mesl span Vojod Darad Ke Betori Ast Ke Faghat Ta Jayi Ke Niaz Darad Faza Eshghal Mikonad Va Baraks block ke nmizarad hich element kenarash Beshinad inline migzarad Vali Nmitavan Be On Yek Width Dad.

    3. inline-block: In Value Mananad inline Kar Mikonad Ama Ba Ye Tafavot Ke Be On Mitonim Yek Width Moshakhas Bedahim Va Kar Mikonad.

    4. none: In Value Kari Mikonad Ke Engar In Element Aslan Vojod Nadarad Va Be Tor Kol Na Padid Mishavad. Fargh In Value Ba Code visibility:hidden; In Ast Ke Display:none; Kari Mikonad Ke Engar In Element Aslan Vojod Nadashte Va Jaye On Element Ro Khali Nmigzarad Ama visibility none On ro Makhfi mikonad Va Faghat Jaye Onro Khali Migzarad.
